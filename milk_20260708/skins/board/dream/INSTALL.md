# 드림 스킨 (무료판)

페어/드림 아카이브용 milk 게시판 스킨. 페어 카드 갤러리 + 캐릭터 A·B 프로필 미니홈 + 하위 게시판(게시글·메모·커미션·이야기).

## 설치
1. `skins/board/dream/` 폴더를 milk 루트의 `skins/board/` 아래에 업로드
2. 관리자 → 게시판 설정에서 스킨을 **드림**으로 선택
3. 끝. 필요한 테이블은 첫 사용 시 자동 생성됩니다.

## 구성
- **페어 카드 갤러리** — 드림(페어) 목록을 카드로
- **미니홈 view** — 헤더 + 캐릭터 A·B 패널(컬러 팔레트·신상·BGM) + 페어 설정
- **하위 게시판** (미니홈 좌측 메뉴)
  - 게시글 — 카드 목록 → 클릭 시 상세
  - 메모 — 미리보기 목록 → 클릭 시 모달
  - 커미션 — 이미지 갤러리 → 클릭 시 상세
  - 이야기 — 작품 + 회차(소설) 2단계

## 데이터 모델 (milk 표준)
- {post} — 드림 본체 (po_content 에 캐릭터 A·B JSON)
- {dream_post} — 하위 게시글 매칭 (게시글·메모·커미션)
- {dream_novels} / {dream_chapters} — 이야기(작품·회차)
- _dreamlog 게시판 — 하위 게시글 저장소 (자동 생성, 메뉴 비노출)

모든 테이블은 첫 사용 시 자동 생성되며, milk 코어 파일은 수정하지 않습니다.

## 작성 팁
- 작성 폼에서 페어 이름·대표색·캐릭터 A/B(이름·이미지·컬러·신상·설명·BGM) 입력
- 카드 썸네일은 비워두면 헤더 → 캐릭터 A 이미지 순으로 자동 지정
- 비밀글로 설정하면 관리자만 볼 수 있습니다
